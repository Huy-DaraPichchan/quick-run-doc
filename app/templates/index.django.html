Installation

{% comment %}Step 1{% endcomment %}

1. Checking Python version

Before creating a virtual environment, ensure you have a suitable Python interpreter installed and that you can reference it correctly.

python --version # Expected output: Python 3.12.3 or newer
{% comment %}Step 2{% endcomment %}

2. Setting up Python Virtual Environment

It's best to create a new Python virtual environment for each of your project so that you won't have to deal with version conflicts between packages.

1. Create a virtual environment named venv:
python -m venv venv # venv (command) venv (name)
  • Windows (CMD):
    . venv\Scripts\activate
  • macOS/Linux (bash/zsh):
    source venv/bin/activate
2. Confirm which Python you are using:
which python # Should point to your venv/Scripts/python (Windows) or venv/bin/python (macOS/Linux)
{% comment %}Step 3{% endcomment %}

3. Install Django

After activating the virtual environment, install Django using pip.

1. Install Django:
pip install Django
{% comment %}Step 4{% endcomment %}

4. Create Django Project

Use django-admin to start a new project named root.

django-admin startproject root
{% comment %}Step 5{% endcomment %}

5. Navigate into the Project and Start an App

Use django-admin to start a new project named root.

django-admin startproject root
  • Change directory into the project folder:
    cd root
  • Use django-admin and startapp with app as its name:
    django-admin startapp app
{% comment %}Step 6{% endcomment %}

6. Run the app locally

Test run the app locally or listen on all available network interfaces.

  • Run on localhost:
    python manage.py runserver
  • Listen on all available network interfaces.
    python manage.py runserver 0.0.0.0:8000

1. Configure a Basic Index View

a. Create the Template
  1. In your app directory, create a folder templates/.
  2. Inside app/templates/, create index.django.html with desired HTML content, for example: 
                        
<!DOCTYPE html> 
  <html lang="en"> 
  <head> 
    <meta charset="UTF-8"> 
    <title>Home</title> 
  </head> 

  <body> 
    <h1>Welcome to Django</h1> 
  </body> 
</html>

2. Define the URL Configuration for the App

In app/urls.py, add: 

                        
from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
]

3. Create the Index View

In app/views.py, add: 

                        
from django.shortcuts import render

def index(request):
    return render(request, 'index.django.html')

4. Include App URLs in the Project

Open root/urls.py and update: 

                        
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),  
    path('', include('app.urls')),  # <- add here
]

5. Test the Index Page

Restart the development server if it is not running:

python manage.py runserver
